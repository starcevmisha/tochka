##### 2. Добавить денег на счёт
`http://127.0.0.1/api/add?uuid=<uuid-счёта>&amount=<сумма>`

Пример ответа:
```
{
    "addition": {
        "balance": 3000,
        "created": "Sat, 19 Oct 2019 09:53:49 GMT",
        "fio": "oooo",
        "hold": 0,
        "last_update": "Sat, 19 Oct 2019 09:53:49 GMT",
        "status": "Active"
    },
    "description": "OK",
    "result": true,
    "status": 200
}
```
